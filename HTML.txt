O Box Model é um conceito fundamental em HTML e CSS que descreve como os elementos são estruturados e como suas dimensões são calculadas. Cada elemento em uma página web é representado como uma "caixa" (box) composta por quatro partes principais: Content, Padding, Border, Margin.
Box-level: A tag ocupa a linha inteira forma uma caixa e quebra a linha para o proxima linha.Exemplos de elementos box-level:<div>, <p>, <h1> a <h6>, <header>, <footer>, <section> <ul>, <ol>, <li>, <table>
Inline-level:A tag nao quebra linha manten no mesmo lugar nao pode usar o height e width.
Exemplos de elementos inline-level:<span>, <a>, <strong>, <em>, <img>, <input>, <button>
